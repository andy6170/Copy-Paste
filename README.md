This adds Copy and Paste functionality to the Battlefield 6 Portal Editor.

Simply add this URL to the extension within the plugin manager:

https://andy6170.github.io/Copy-Paste/manifest.json

Adds 2 new buttons 
Copy - BF6
Paste - BF6

This is to distinguish between the new and the original function in the base extension.
Suports copying between experiences on different tabs.
